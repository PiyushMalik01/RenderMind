RenderMind 3D Assets Library
============================

Place your 3D models/assets here organized by category.

Supported formats:
- .blend (Blender files)
- .fbx (FBX files)
- .obj (OBJ files) 
- .gltf / .glb (GLTF files)
- .stl (STL files)

Recommended folder structure:
assets/
  furniture/
    chair.blend
    table.fbx
    sofa.obj
  vehicles/
    car.blend
    bike.gltf
  nature/
    tree.blend
    rock.obj

Naming tips:
- Use descriptive names (e.g., "office_chair.blend" not "model1.blend")
- Use underscores instead of spaces
- Keep names lowercase for easier matching
